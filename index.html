第一章 绪论

算法+数据结构=程序

  • 数据结构：相互之间存在一定关系的数据元素的集合

    • 数据的逻辑结构：数据元素之间逻辑关系的整体

      • 分为集合（无关系），线性结构（一对一），树结构（一对多），图结构（多对多）
      • 是用户视图，面向问题

    • 数据的存储结构：数据及其逻辑结构在计算机（内存）中的表示

      • 分为顺序和链状
      • 是实现视图，面向计算机

  • 算法：算法就是求解问题的方法和步骤。经过一系列操作步骤能够输出，不一定会有输入

Git使用笔记

发表于 2022-08-22 分类于 笔记 本文字数： 396
Command Usage Args
git init 初始化一个空的仓库
git add 把文件从工作区添加至暂存区 <filename> 文件名
-A 所有文件
git commit 将暂存区的更改同步至本地仓库 -m <message> 提交注释(必填)
git push 将本地仓库的内容推送至远程仓库 -u <upstream> <branch> 远程仓库分支
git clone 复制仓库 <repo>仓库git地址
git diff 对比差异
git branch 仓库分支 -add <name>添加分支
-m 移动
git switch 切换分支 <branch>仓库分支
git fetch 从远程仓库拉取到本地仓库
git pull 从远程仓库拉取到工作区
git merge 合并分支
git remote add 添加远程仓库 <name> <repo>仓库名 仓库git地址

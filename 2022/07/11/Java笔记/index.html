2.数据类型

类型 空间 范围 备注
byte 8 -2^7~2^7-1 8位=1byte
short 16 -2^15~2^15-1
int 32 -2^31~2^31-1
long 64 -2^63~2^63-1 数据末尾+L\l
float 32 较大 数据末尾+F\f
double 64 很大 数据末尾+D\d(默认)
char 16 0~2^16 用’ ‘表示，使用Unicode
boolean 0,1 不能进行类型转换

二、字符串String

1.声明方式

2.常用方法

1.length()

  • 返回长度

2.indexOf(string)

  • 返回子串string第一次出现的位置(下标)，否则返回-1

3.lastIndexOf(string)

  • 返回子串string最后一次出现的位置(下标)，否则返回-1

4.charAt(int)

  • 返回int位置的字符

5.substring(int start,int end)

  • 一个参数：返回(包含)start位置之后的子串
  • 两个参数：返回从start位置(包含)到end位置(不含)的子串

6.trim()

  • 去掉空格

7.replace(char old,char new)

  • 将字符串里所有old替换成new后返回，无替换返回原串

8.startsWith(string)

  • 判断是否以string开头，返回Boolean

9.endsWith(string)

  • 判断是否以string结尾，返回Boolean

10.equals(string)

  • 判断字符串与string是否长度相等且字符相等，返回Boolean

11.equalsIgnoreCase(string)

  • 效果同上，但忽略大小写

12.compareTo*=(string)

  • 将字符串与string按Unicode字典序比较，主字符串在string前输出负数，反之正数

13.toLowerCase()

  • 转换成小写

14.toUpperCase()

  • 转换成大写

15.spilt(string,int)

  • 用于分割字符串，返回一个数组，string为分隔符，可以用|定义多个分隔符如”,|.”，或是使用正则表达式
  • int为分割次数上限，可以省略

16.String.format(String format,Object arg)

  • format是特定的格式字符串




  • object是待处理的对象

17.matches(string reg)

  • 匹配正则表达式，返回Boolean

2.常用方法

1.append(arg)

  • 在字符串末尾追加内容，可以是任何数据类型或对象

  • 删除字符串中从start(包含)到end(不含)的内容

三、数组Array

1.声明方式

2.常用方法

1.length

  • 返回长度

2.fill(int[] a,int start,int end,int value)

  • 把从start位置(包含)到end位置(不含)的元素替换为value
  • 只有a和value参数：把数组中的所有元素替换为value
  • start等于end时什么都不做

3.Arrays.sort(array)

  • 按字典序排序

4.copyOf(array,int)

  • 复制一个数组，数组长度为int
  • 长度小于arr.length则将多余的截去
  • 长度大于arr.length，多余部分int填0，char填null

5.copyOfRange(array,int start,int end)

  • 把从start位置(包含)到end位置(不含)的元素复制到新的数组

6.binarySearch(array,key,start,end)

  • 使用二分查找搜索数组从start位置(包含)到end位置(不含)的元素，返回下标
  • 找不到返回-1或“-(插入点)”(例如在1,2,4中寻找3,4的下标是2，3在4的左边，所以返回-2)
  • 使用前应对数组进行排序，否则结果未知，有多个相同值也无法确定
  • start与end可以省略

四、类与对象

1.修饰符

public protected private
本类 √ √ √
同包的类或子类 √ √
其他包的类或子类 √

2.this

  • 成员方法中的this指的是类对象本身

3.静态static

  • 静态方法中不能使用this

  • 静态方法中不能调用非静态方法

  • 不能在方法中声明静态变量

声明静态变量的一种方法

4.类的主方法

  • 主方法是静态的，其调用的方法也要是静态的
  • 主方法没有返回值
  • 主方法的参数是个数组，由args[n]获取；个数可以由args.length获取
  • 主方法的参数可以由编辑器设置

5.构造

  • 是一个public方法，与类名同名，可以有多个不同的重载
  • 创建对象时使用new

6.对象的比较

  • 以string为例

    • 使用==比较：比较的是对象引用的地址

    • 使用equals()比较：比较的是内容

7.对象的销毁

  • 销毁方法

    • 超过作用域

    • 将对象赋值为null

  • 只能销毁由new创建的对象

  • 类内使用protected void finalize()方法，类似cpp中的析构函数

  • 因对象销毁时间未知，所以finalize()函数不一定执行，可以使用System.gc()强制启动回收来销毁对象

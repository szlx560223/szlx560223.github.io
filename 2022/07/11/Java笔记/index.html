2.数据类型

类型 空间 范围 备注
byte 8 -2^7~2^7-1 8位=1byte
short 16 -2^15~2^15-1
int 32 -2^31~2^31-1
long 64 -2^63~2^63-1 数据末尾+L\l
float 32 较大 数据末尾+F\f
double 64 很大 数据末尾+D\d(默认)
char 16 0~2^16 用’ ‘表示，使用Unicode
boolean 0,1 不能进行类型转换

3.转义符号

转义 意义
\ddd 八进制数表示的字符
\uxxxx 十六进制数表示的字符
\‘ ‘
\\ \
\t 制表符
\r 回车
\n 换行
\b 退格
\f 换页

  • 变量声明与cpp相同

  • 静态声明static

  • <<左移左侧舍弃，右侧补零，>>右移右侧舍弃，左侧按原最左位补数

二、字符串String

1.声明方式

2.常用方法

1.length()

  • 返回长度

2.indexOf(string)

  • 返回子串string第一次出现的位置(下标)，否则返回-1

3.lastIndexOf(string)

  • 返回子串string最后一次出现的位置(下标)，否则返回-1

4.charAt(int)

  • 返回int位置的字符

5.substring(int start,int end)

  • 一个参数：返回(包含)start位置之后的子串
  • 两个参数：返回从start位置(包含)到end位置(不含)的子串

6.trim()

  • 去掉空格

7.replace(char old,char new)

  • 将字符串里所有old替换成new后返回，无替换返回原串

8.startsWith(string)

  • 判断是否以string开头，返回Boolean

9.endsWith(string)

  • 判断是否以string结尾，返回Boolean

10.equals(string)

  • 判断字符串与string是否长度相等且字符相等，返回Boolean

11.equalsIgnoreCase(string)

  • 效果同上，但忽略大小写

12.compareTo*=(string)

  • 将字符串与string按Unicode字典序比较，主字符串在string前输出负数，反之正数

13.toLowerCase()

  • 转换成小写

14.toUpperCase()

  • 转换成大写

15.spilt(string,int)

  • 用于分割字符串，返回一个数组，string为分隔符，可以用|定义多个分隔符如”,|.”，或是使用正则表达式
  • int为分割次数上限，可以省略

16.String.format(String format,Object arg)

  • format是特定的格式字符串




  • object是待处理的对象

17.matches(string reg)

  • 匹配正则表达式，返回Boolean

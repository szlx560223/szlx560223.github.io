Git使用笔记

发表于 2022-08-22 分类于 笔记 阅读次数： 本文字数： 396
Command Usage Args
git init 初始化一个空的仓库
git add 把文件从工作区添加至暂存区 <filename> 文件名
-A 所有文件
git commit 将暂存区的更改同步至本地仓库 -m <message> 提交注释(必填)
git push 将本地仓库的内容推送至远程仓库 -u <upstream> <branch> 远程仓库分支
git clone 复制仓库 <repo>仓库git地址
git diff 对比差异
git branch 仓库分支 -add <name>添加分支
-m 移动
git switch 切换分支 <branch>仓库分支
git fetch 从远程仓库拉取到本地仓库
git pull 从远程仓库拉取到工作区
git merge 合并分支
git remote add 添加远程仓库 <name> <repo>仓库名 仓库git地址
